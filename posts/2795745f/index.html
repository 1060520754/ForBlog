1.Java 程序入门
发表于 2021-04-06| 更新于 2021-04-08
阅读量:
起源
  1. Java诞生于1995年 SUN公司
  2. Java之父：詹姆斯·高斯林
  3. Java三大版本：
    1. J2SE 标准版
    2. J2ME 微型版
    3. J2EE 企业版
特点
  1. 开源
  2. 简单
  3. 安全
  4. 跨平台
  5. 垃圾自动回收
  6. 面向对象
JDK、JRE、JVM
  1. JDK (Java Development Kit java开发工具包)
  2. JRE (Java Runtime Environment java运行工具包)
  3. JVM (Java Virtual Machine java虚拟机)
  4. JDK = JRE + 工具包 + 类库
  5. JRE = JVM + 解释器
环境变量
  1. 变量名：JAVA_HOME

​ 变量值：JDK文件路径

  1. 变量名：Path

​ 变量值： JDK下的bin文件路径

配置环境变量的意义：让系统在任意路径下都能执行被配置的程序

第一个Java程序

java源文件后缀名为 .java 经过编译之后 转为字节码文件，后缀名为 .class 然后可以解释运行

注释

  1. 单行注释： //

  2. 多行注释： /* */

  3. 文档注释： /** */
